# Перевод игры Picture Shooter

## Структура репозитория

- template.pot - Шаблон перевода. **Не редактируйте этот файл.**
- src/ - Папка, в которой содержатся **.po** файлы.

##   Создание переводов



1. Скачайте или склонируйте этот проект.

2. Для работы с **.po** вам нужно установить програму [Poedit](https://poedit.net/download)

3. Создайте новый файл перевода от шаблона template.po. Переведите все ключи и сохраните .po файл в папку src.

4. Отправьте пул реквест.



## Правила оформления переводов



Весь текст на кнопках, на которые может нажать игрок, а также заголовки и подзаголовки (например, ключи с LABEL в названии) должны быть написаны КАПСОМ.

Все ключи, которые имееют LABEL в названии должны подчеркиваться символом "-" на следующей строчке. Например, "VIDEO \\n -------------".



Также рекомендуется чаще смотреть в файлы перевода от разработчиков, en.po и ru.po.
